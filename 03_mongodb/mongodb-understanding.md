## mongodb คืออะไร
mongodb คือรูปแบบการเก็บข้อมูลแบบก้อนเอกสาร document โดยใช้รูปแบบการเก็บภาษา bson คล้ายหรือเหมือน javascript เรียกว่า mql

## mongodb ทำงานอย่างไร
โครงสร้างการเชื่อมต่อ การเก็บ การเรียกดูของ mongodb ไม่ได้ต่างจาก postgre มาก แต่ต่างกันตรงที่การใช้งาน server ของ mongodb นั้น ใช้ paas หลักของ mongodb atlas เองเลย โดยเราไม่ต้องเป็น server ให้แบบ postgresql

การสร้าง เรียกใช้ บันทึก หรือลบ ข้อมูลใน database เราใช้ภาษา javascript ในการทำ

โดยโปรแกรมที่จะใช้เป็น interface ติดต่อกับ mongodb ก็เป็น extension ของ vscode โดย paas ของ mongodb atlas มีความสามารถในการตั้งค่าที่จำเป็นครบทั้งหมด เช่น IP whitelist / user access segmentation

## mongodb มีข้อดีอย่างไร
หลักโครงสร้างของ mongodb อยู่ฝั่งตรงข้ามกับ sql หมายถึงเราสามารถเก็บข้อมูลในรูปแบบ bson โดยที่ไม่ต้องมี ตารางหรือ schema มารองรับ แต่เราสามรถเก็บค่าของ feild ได้ต่างๆกัน โดยจะเขียนคำสั่ง feild เหล่านี้มาใช้ในภายหลัง
