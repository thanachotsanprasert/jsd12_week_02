## postgresql คืออะไร
postgresql คือระบบการจัดเก็บข้อมูลแบบ relational โดยใช้ภาษา SQL

## postgresql ทำงานอย่างไร
ในการสร้างเว็บไซด์หรือ application ต้องมีสถานที่ในการเก็บข้อมูล ซึ่งเราเรียกว่า database ซึ่ง postgresql คือประเภทหนึ่งของ database นี้

การสร้าง เรียกใช้ บันทึก หรือลบ ข้อมูลใน database เราใช้ภาาา sql ในการทำ

ในการเรียนนี้เราเลือกใช้ server ฟรี จาก aiven ซึ่งเป็น paas ที่ให้พื้นที่ข้อมูลฟรี และระบบการจัดการข้อมูลของ postgresql ได้ฟรี

โดยโปรแกรมที่จะใช้เป็น interface ติดต่อกับ database ก็มีมากมายต่างกันไป ซึ่งในการเรียนนี้เราใช้ postgresql ที่ลงโดย brew และใช้ postgresql extension ใน vscode เป็น cli หรือ gui ในการเขียน sql ทำงานกับ postgresql server ของ aiven และ postgresql ที่ลงโดย brew ซึ่งก็มีโปรแกรมอื่นๆอีกที่ใช้ได้เช่น Dbeaver

## sql ทำงานอย่างไีร
ก่อนจะอธิบายว่า postgresql มีข้อดีอย่างไร ก็จำเป็นต้องอธิบายว่า sql ทำงานยังไง
ในการเริ่มใช้ sql หรือการวางโครงสร้างของ databse ผู้ใช้ต้องวางแผนตารางหรือ schema ก่อน ซึ่งในการวางแผน schema หรือตาราง ให้นึกถึงการใช้จริงใน operation โดยต้องวางแผนว่าตารางแต่ละอันมีหน้าที่เผื่ออะไร และจะเชื่อมต่อกันแบบไหน

ข้อมูลใดๆที่ไม่มีตารางหรือ schema รองรับจะทำให้ใช้หรือเก็ยข้อมูลนั้นใช้ไม่ได้ อีกมุมหนึ่งคือ sql ทีความเฉพาะเจาะจงและความเป็นแบบแผนสูง ดังนั้นความผิดพลาดจะน้อย แต่ยุ่งยากในการวางแผน และการใช้

## postgresql มีข้อดีอย่างไร
อย่างที่กล่าวมาข้างต้น ข้อดีของ postgresql คือความเป็นระเบียบ แบบแผน การลด หรือป้องกันความผิดพลาดที่ต้องแลกมากับความยากในการ scale หรือแก้ไขโครงสร้าง
